# 3D Carousel – Fusion 360 Project

## Overview

This project represents a 3D carousel modeled in Autodesk Fusion 360. The carousel is designed as an amusement park ride with a circular rotating structure and decorative horses.

## Description

The model contains the main parts of a classic carousel, including the base platform, central support column, top canopy, vertical poles, and horses. The carousel is designed to suggest circular motion, while the horses are placed on poles to represent the traditional up-and-down movement of a real carousel.

## Main Components

- Base platform
- Central column
- Rotating circular platform
- Top canopy
- Vertical poles
- Carousel horses
- Decorative elements

## Motion Concept

The carousel is intended to rotate around its central axis. At the same time, the horses move vertically on their poles, creating a realistic carousel animation effect.

## Mechanical Actuation

The carousel is designed to be mechanically driven using a manual crank (handle). The motion is transmitted through a simple mechanism:

Handle (crank) → horizontal shaft → gear system → central vertical axis → rotating platform

By rotating the handle, the motion is transferred through a set of gears to the main axis of the carousel, causing the entire platform to spin. This setup simulates a realistic mechanical drive system, similar to traditional manually operated machines.

Additionally, the vertical movement of the decorative elements is conceptually based on a cam-like mechanism, which allows them to move up and down while the carousel rotates.

This approach demonstrates basic mechanical principles such as rotational motion, transmission, and synchronization of multiple moving parts.

## Software Used
